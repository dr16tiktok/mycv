# Skill: UI Consistency (Wizard)

Este skill existe para que cualquiera que trabaje en el repo respete el mismo formato visual y de navegación.

## Checklist
- ¿Es 1 pregunta por pantalla?
- ¿Botones fijos abajo (Saltar / Siguiente)?
- ¿Tipografía grande y legible en mobile?
- ¿Espaciado consistente?
- ¿Animaciones sutiles (Framer Motion) sin exagerar?

## No hacer
- No agregar pantallas con layouts diferentes.
- No introducir múltiples inputs complejos en una sola pantalla.
