# Screen Format — Wizard

**Objetivo:** todas las pantallas del wizard deben verse iguales (estructura fija) y solo cambia el contenido.

## Layout base
```
[Title]
[Subtitle]
[Input/Options]

(espacio flexible)

[Skip] [Next]
```

## Reglas
- Botonera fija abajo.
- 1 input o 1 grupo de opciones máximo.
- Validación inline simple (si aplica).
- CTA principal siempre a la derecha.
